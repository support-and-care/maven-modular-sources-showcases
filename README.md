# Java Modules Showcases with Apache Maven 4

This repository contains companion code for blog articles about Java Modules (Java Platform Module System) with Maven 4.

## Blog Articles

- [Blog 1: Basics](blog/blog-1-basics.md) - Introduction to Java Modules with Maven 4
- [Blog 1: Homework](blog/blog-1-homework.md) - Solving resource and packaging challenges

Each blog article has a corresponding branch with the code at that state:

| Article | Branch | Code Changes |
|---------|--------|-------------|
| Blog 1: Basics | `main` | Initial modular project setup |
| Blog 1: Homework | `blog-1-homework` | Resource and JAR plugin workarounds |

See the blog articles for build instructions, project structure, and how to run the examples.
